Projects

Machine Learning

DiffEdit - Text Guided Image Editing with Diffusion

Combined DiffEdit, a text-conditioned diffusion models for semantic image editing, with BLIP and other text-to-image models to create a end-to-end generation and editing pipeline.

Evaluation of Fairness and Factualness with LLM

Developed a zero-shot evaluation framework using Microsoft Phi-2 and Mistral-7B, incorporating GPT-generated evidence and chain-of-thought reasoning to assess fairness and factuality in textual claims, achieving a 76% accuracy rate.

EEG Signal Classification

Compared CNN, RNN, Transformer, and hybrid models on EEG signal analysis while sweeping hyper-parameters, time-bin resolutions, and augmentation.

Software Development

Job Search

A personalized job recommendation platform that suggests opportunities based on user interactions. It features a full-stack architecture with Java-based REST APIs, a responsive frontend, and a recommendation engine leveraging keyword extraction and user history. The web app uses Amazon RDS with Redis for caching and persistence.

Bruin O Bruin

Led a team of 4 to develop a web-based elimination gama built with Node.js, React, and SQLite, featuring randomized challenges, live discussion forums, and a real-time leaderboard; deployed on Azure, it engaged over 50 % of the class.

Sorting Visualizer

Developed an interactive sorting algorithm visualizer using Vite, React, TypeScript, and Tailwind CSS. The tool features real-time animations of algorithms like Bubble Sort, synchronized with pseudocode display.
